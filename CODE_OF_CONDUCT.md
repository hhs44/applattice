# 社区行为准则

## 我们的承诺

我们致力于为所有参与者提供友好、安全、专业且无骚扰的协作环境，不因经验、身份、背景或观点而区别对待。

## 期望行为

- 对人保持尊重，对方案进行具体、可验证的讨论；
- 接受建设性反馈，并在犯错后主动修正；
- 关注项目和社区的共同利益；
- 对安全、隐私、知识产权和他人时间保持审慎；
- 在意见不一致时说明事实、约束和权衡，避免人身攻击。

## 不可接受行为

- 骚扰、威胁、歧视、侮辱或煽动性言论；
- 未经允许公开他人的私人信息；
- 持续打断讨论、恶意灌水或故意破坏协作；
- 以安全报告为名公开可利用细节；
- 其他在专业协作环境中明显不适当的行为。

## 执行

维护者可以编辑、隐藏或删除不符合本准则的内容，并根据严重程度发出警告、临时限制或永久禁止参与。报告行为问题时，请联系仓库维护者并使用可用的私密渠道；不要在公开 Issue 中披露当事人的私人信息。维护者应尽量保护报告者身份，并以一致、善意的方式处理事件。

本准则适用于项目仓库、Issue、Pull Request、讨论区以及代表本项目参与的其他公开空间。
